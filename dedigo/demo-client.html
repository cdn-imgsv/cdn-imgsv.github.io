Salutations,

Nous tenons tous d'abord à vous remercier pour l'achat de votre service web chez nous, nous espérons que votre expérience chez nous sera le meilleur possible, nous vous rappelons que nous disposons de plusieurs moyens de support en cas de besoin.

Nous disposons de plusieurs offres web et nous allons vos éguillez dans le départ de ses deux types d'offres de manière rapide et efficace.

  • Dans le cas d'un hébergement web,
    • Il faut savoir qu'actuellement sur votre hébergement web vous disposez de ce fichier appeler "index.html", merci de le supprimer ou renommer pour que votre index.php ou index.html soit actif.

    Accédez au cPanel, WebMail.
  • Dans le cas d'un revendeur web,
    • Il faut savoir qu'actuellement sur votre espace web que vous disposez pour le site vitrine de votre hébergement web vous avez de ce fichier appeler "index.html", merci de le supprimer ou renommer pour que votre index.php ou index.html soit actif.

    Accédez au/a cPanel, WHM, WebMail.
  • Comment connaître mes accès ?
    • Pour connaître vos accès il suffit de regarder l'e-mail² envoyez par notre service afin de connaître l'hôte ftp, smtp, votre identifiant, mot de passe et autres si vous souhaitez ne pas forcément vous connecter à l'espace client pour une connexion au cPanel.

    ²: Si vous arrivez pas à voir l'e-mail sur votre boîte mail, regarder celle de votre espace client grâce à ce tutoriel.

    Cordialement,
    - Votre équipe DediGo.
